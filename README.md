# AI-Quant-Trading-System
Advanced AI Quant Trading System with machine learning models, smart money concepts, liquidity analysis, and automated trading strategies for multi-asset markets.

## Features

- AI Ensemble Model (RandomForest, XGBoost, LightGBM)
- Smart Money Concepts
- Liquidity Sweep Detection
- Order Block Detection
- Institutional Liquidity Analysis
- Market Structure (BOS / CHOCH)
- Multi Pair Trading
- Dynamic Risk Management
- News Filter
- Strategy Optimization
- Self-Learning Trade Analysis
- Trading Dashboard

## Supported Markets

- XAUUSD
- EURUSD
- GBPUSD
- USDJPY
- AUDUSD
- USDCAD
- USDCHF
- NZDUSD
