# AI Quant Trading System

An advanced AI-based trading bot designed to analyze financial markets using machine learning, smart money concepts, liquidity analysis, and automated trading strategies.

## Features

* AI Ensemble Model (RandomForest, XGBoost, LightGBM)
* Smart Money Concepts
* Liquidity Sweep Detection
* Order Block Detection
* Institutional Liquidity Analysis
* Market Structure (BOS / CHOCH)
* Multi Pair Trading
* Dynamic Risk Management
* News Filter
* Strategy Optimization
* Self-Learning Trade Analysis
* Trading Dashboard

## Supported Markets

* XAUUSD
* EURUSD
* GBPUSD
* USDJPY
* AUDUSD
* USDCAD
* USDCHF
* NZDUSD

## Installation

pip install -r requirements.txt

## Run Trading Bot

python bot.py

## Backtesting

python backtest.py
